# Toolhub

### What is [Toolhub](https://toolhub-ab219.web.app/)?
**Toolhub** is a web application created for Thammasat University students to book and manage the borrowing of engineering tools.
It provides a streamlined digital interface containing three main functions:
- Booking (การจองอุปกรณ์): Allows users to select desired tools (e.g., Electric Drill, Pliers), set specific borrow and return dates/times, and submit the request using their TU ID.
- Status (สถานะการยืม): Allows users to track their current and past borrowing transactions by entering their Username, showing the item, dates, quantity, and status (e.g., pending return).
- Policy (ข้อกำหนดการยืม): Explains the rules for broowing tools from this project.

ToolHub simplifies the logistics of accessing essential project tools for the university's engineering community.
